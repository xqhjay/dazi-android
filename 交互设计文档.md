## 2. 核心页面交互流程

### 2.1 练习流程

```
[练习首页]
    │
    ▼ 选择练习类型
[练习设置页]
    │
    ▼ 配置参数，点击开始
[练习进行页] ◄──────┐
    │                │
    ▼ 完成或退出     │ 下一题
[练习结果页]         │
    │                │
    ▼ 点击继续练习 ──┘
```

### 2.2 错字本流程

```
[练习进行页] ──错误输入──► [自动加入错字本]
                                    │
[练习首页] ──点击错字本──► [错字本列表页]
                                    │
    ▲                               ▼
    └────────── 开始练习 ──── [错字练习页]
```

## 3. 页面详细设计

### 3.1 练习首页

**布局：**
- 顶部：App标题 + 今日练习时长
- 中部：4个功能卡片（2×2网格）
  - 单字练习
  - 词组练习
  - 句子练习
  - 错字本（显示待复习错字数量角标）
- 底部：导航栏

**交互：**
- 卡片点击：跳转到对应设置页
- 错字本角标：实时显示未复习错字数

### 3.2 练习进行页（核心页面）

**布局（从上到下）：**
1. **顶部栏**
   - 返回按钮
   - 练习类型标题
   - 进度（当前/总数）

2. **统计栏**
   - WPM: 实时字/分钟
   - 准确率: 百分比
   - 用时: 00:00格式

3. **目标文本区**
   - 大号字体显示目标文本
   - 已输入正确：绿色
   - 当前输入：光标闪烁
   - 已输入错误：红色背景
   - 未输入：灰色

4. **输入反馈区**
   - 当前字的拆字可视化（可选显示）
   - 编码提示（可选显示）

5. **虚拟键盘区**
   - 标准键盘布局
   - 当前需要按的键高亮提示
   - 错误按键震动反馈

**交互细节：**
- 键盘输入：即时响应
- 错误输入：
  - 目标字符闪烁红色
  - 显示正确字根提示（2秒后自动消失）
  - 记录错误
- 正确输入：
  - 目标字符变绿色
  - 自动推进到下一字符
- 退格：支持回退修改
- 完成：弹出结果页

### 3.3 拆字可视化组件

**触发：** 用户输入错误或长按目标字

**展示内容：**
```
┌─────────────────┐
│  想              │
│  xiǎng          │
├─────────────────┤
│  拆字: 木 + 目 + 心  │
│  编码: SHNU      │
│                 │
│  [木]─S  [目]─H  [心]─N  │
│  末笔识别码: U    │
│  简码: 二级简码 SH │
└─────────────────┘
```

**动画：**
- 字根依次高亮显示
- 对应键位在键盘上闪烁

### 3.4 字根表页

**布局：**
- 顶部：区位筛选标签（全部/横区/竖区/撇区/捺区/折区）
- 中部：键盘布局图
  - 25个键位按钮
  - 每个键位显示主要字根
- 点击键位：底部弹窗显示该键所有字根

**交互：**
- 键位点击：弹出详情
- 字根点击：显示例字
- 搜索：支持按字根或例字搜索

### 3.5 练习结果页

**布局：**
- 顶部：成绩等级（S/A/B/C/D）
- 成绩卡片：
  - WPM
  - 准确率
  - 用时
  - 正确字数/总字数
- 详细统计：
  - 退格次数
  - 错误字数
  - 平均反应时间
- 错字列表：可点击加入错字本
- 操作按钮：
  - 再来一次
  - 返回首页
  - 分享成绩

## 4. 交互规范

### 4.1 反馈规则

| 场景 | 反馈方式 | 持续时间 |
|------|----------|----------|
| 输入正确 | 字符变绿 + 轻微音效 | 即时 |
| 输入错误 | 字符变红 + 震动 + 提示 | 2秒 |
| 完成练习 | 庆祝动画 + 成绩页 | - |
| 加入错字本 | Toast提示 | 2秒 |

### 4.2 动画规范

| 动画 | 时长 | 缓动 |
|------|------|------|
| 页面切换 | 300ms | ease-in-out |
| 字符状态变化 | 100ms | ease-out |
| 拆字提示弹出 | 200ms | spring |
| 键盘按键按下 | 50ms | ease-out |
| 成绩数字滚动 | 800ms | ease-out |

### 4.3 手势操作

| 手势 | 功能 |
|------|------|
| 左滑 | 返回上一页 |
| 长按目标字 | 显示拆字详情 |
| 双击键盘区 | 隐藏/显示键盘 |

## 5. 状态管理

### 5.1 练习状态机

```
[空闲] --开始练习--> [进行中]
[进行中] --暂停--> [暂停]
[暂停] --继续--> [进行中]
[进行中] --完成--> [完成]
[进行中] --退出--> [空闲]
[完成] --再来一次--> [进行中]
[完成] --返回--> [空闲]
```

### 5.2 输入状态

```
[等待输入] --正确字符--> [等待输入]
[等待输入] --错误字符--> [错误提示] --2秒--> [等待输入]
[等待输入] --退格--> [等待输入]
[等待输入] --完成--> [完成]
```

## 6. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 码表数据缺失 | 显示默认提示，记录日志 |
| 练习中断（来电等） | 自动暂停，恢复时继续 |
| 存储空间不足 | 提示清理缓存 |
| 导入文件格式错误 | 提示正确格式示例 |
