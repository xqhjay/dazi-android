# 形码大师 — 用户故事文档

## 用户画像

###  persona 1: 小明 — 形码初学者
- **背景**：大学生，想学习五笔输入法提高打字效率
- **痛点**：字根难记，拆字规则不清楚，不知道从何练起
- **目标**：系统学习字根，掌握基本拆字规则

### persona 2: 小红 — 进阶练习者
- **背景**：已学会五笔，但速度停留在60字/分钟
- **痛点**：某些字总是想不起来怎么拆，缺乏针对性练习
- **目标**：突破速度瓶颈，达到100字/分钟以上

### persona 3: 老张 — 专业录入员
- **背景**：从事文字录入工作多年
- **痛点**：需要保持输入速度，偶尔有生疏的字
- **目标**：保持速度，快速复习生疏字

---

## 用户故事列表

### US-001 字根学习
**作为** 形码初学者  
**我希望** 按字根区域（横竖撇捺折）系统学习  
**以便** 掌握字根在键盘上的分布

**验收标准：**
- [ ] 可选择单一区域或多个区域组合练习
- [ ] 每个字显示对应的键位和字根图
- [ ] 错误时提示正确字根
- [ ] 练习进度可视化

**优先级：** P0

---

### US-002 单字拆字练习
**作为** 初学者  
**我希望** 练习时能看到汉字的拆字过程  
**以便** 理解拆字规则，加深记忆

**验收标准：**
- [ ] 输入时动态展示字根拆分动画
- [ ] 显示每个字根对应的键位
- [ ] 显示末笔识别码（如有）
- [ ] 支持简码提示

**优先级：** P0

---

### US-003 看打练习
**作为** 进阶练习者  
**我希望** 对照文本进行连续输入练习  
**以便** 提升实际输入速度

**验收标准：**
- [ ] 清晰显示目标文本
- [ ] 已输入字符实时比对，正确绿色/错误红色
- [ ] 支持退格修改
- [ ] 实时显示WPM和准确率
- [ ] 练习结束显示详细统计

**优先级：** P0

---

### US-004 错字自动收集
**作为** 练习者  
**我希望** 系统能自动记录我常错的字  
**以便** 后续针对性复习

**验收标准：**
- [ ] 练习中输入错误的字自动加入错字本
- [ ] 同一字多次错误累计计数
- [ ] 错字本支持查看、删除、排序
- [ ] 支持从错字本一键发起练习

**优先级：** P0

---

### US-005 字根查询
**作为** 学习者  
**我希望** 随时查询任意汉字的字根和编码  
**以便** 遇到不会的字快速查找

**验收标准：**
- [ ] 支持按汉字查询编码和拆字
- [ ] 支持按编码反查汉字
- [ ] 支持按拼音查询
- [ ] 显示完整的拆字过程

**优先级：** P0

---

### US-006 交互式字根表
**作为** 初学者  
**我希望** 有一个交互式的字根表  
**以便** 直观学习字根分布

**验收标准：**
- [ ] 按键盘布局展示25个键位的字根
- [ ] 点击键位显示该键所有字根
- [ ] 支持按区位筛选（横区G-A、竖区H-M等）
- [ ] 支持按笔画数筛选

**优先级：** P0

---

### US-007 练习统计
**作为** 进阶练习者  
**我希望** 查看我的练习历史和进步趋势  
**以便** 了解自己的提升情况

**验收标准：**
- [ ] 记录每次练习的时间、类型、速度、准确率
- [ ] 显示最近7天/30天的练习趋势
- [ ] 显示最佳成绩
- [ ] 显示总练习时长和总字数

**优先级：** P1

---

### US-008 多码表支持
**作为** 用户  
**我希望** 支持五笔86、五笔98等不同码表  
**以便** 选择我学习的版本

**验收标准：**
- [ ] 支持五笔86版切换
- [ ] 支持五笔98版切换
- [ ] 切换后所有功能使用对应码表
- [ ] 码表数据离线可用

**优先级：** P0

---

### US-009 深色模式
**作为** 长时间练习的用户  
**我希望** 有深色模式  
**以便** 减少视觉疲劳

**验收标准：**
- [ ] 支持浅色/深色主题切换
- [ ] 跟随系统主题设置
- [ ] 所有界面适配深色模式

**优先级：** P0

---

### US-010 自定义练习内容
**作为** 进阶用户  
**我希望** 能导入自己的练习文本  
**以便** 练习实际工作中需要的 content

**验收标准：**
- [ ] 支持粘贴文本创建自定义练习
- [ ] 支持从文件导入文本
- [ ] 支持保存自定义练习内容

**优先级：** P1

---

### US-011 节奏训练（V1.1）
**作为** 希望突破速度瓶颈的用户  
**我希望** 有节奏训练模式  
**以便** 培养连续输入的节奏感

**验收标准：**
- [ ] 类似音乐游戏的节奏条
- [ ] 强制跟打，错过则中断
- [ ] 逐渐加快速度

**优先级：** P2

---

### US-012 对战模式（V1.1）
**作为** 喜欢竞争的用户  
**我希望** 有对战或排行榜功能  
**以便** 增加练习动力

**验收标准：**
- [ ] 支持1v1对战
- [ ] 排行榜显示
- [ ] 练习成绩可分享

**优先级：** P2

---
