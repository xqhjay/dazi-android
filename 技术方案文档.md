## 4. 数据库设计

### 4.1 CharacterEntity（汉字码表）

```kotlin
@Entity(tableName = "characters")
data class CharacterEntity(
    @PrimaryKey val id: Int = 0,
    val char: String,           // 汉字
    val wubi86: String,         // 五笔86编码
    val wubi98: String,         // 五笔98编码
    val pinyin: String,         // 拼音
    val radical: String,        // 部首
    val strokeCount: Int,       // 笔画数
    val zone: String,           // 字根区域 (G-A横区, H-M竖区, T-Q撇区, Y-P捺区, N-X折区)
    val breakdown: String,      // 拆字JSON (如: ["木","目","心"])
    val isLevel1Short: Boolean, // 是否一级简码
    val isLevel2Short: Boolean, // 是否二级简码
    val frequency: Int          // 使用频率
)
```

### 4.2 PracticeRecordEntity（练习记录）

```kotlin
@Entity(tableName = "practice_records")
data class PracticeRecordEntity(
    @PrimaryKey(autoGenerate = true) val id: Long = 0,
    val timestamp: Long,
    val type: String,           // 练习类型 (single/phrase/sentence)
    val duration: Long,         // 练习时长(ms)
    val totalChars: Int,        // 总字符数
    val correctChars: Int,      // 正确字符数
    val wrongChars: Int,        // 错误字符数
    val backspaceCount: Int,    // 退格次数
    val wpm: Float,             // 字/分钟
    val accuracy: Float         // 准确率
)
```

### 4.3 MistakeBookEntity（错字本）


## 5. 核心算法

### 5.1 WPM计算

```kotlin
fun calculateWPM(charCount: Int, durationMs: Long): Float {
    val minutes = durationMs / 60000.0
    return if (minutes > 0) (charCount / minutes).toFloat() else 0f
}
```

### 5.2 准确率计算

```kotlin
fun calculateAccuracy(correct: Int, total: Int): Float {
    return if (total > 0) (correct.toFloat() / total * 100) else 0f
}
```

## 10. 性能优化策略

1. **码表加载**：应用启动时异步加载，使用Room缓存
2. **拆字数据**：按需加载，缓存常用字
3. **Compose优化**：使用remember/derivedStateOf减少重组
4. **数据库**：使用索引优化查询，分页加载历史记录
5. **包体积**：码表数据使用JSON压缩，ProGuard混淆

## 11. 安全设计

1. 用户数据本地加密存储（EncryptedSharedPreferences）
2. 码表数据只读，防止篡改
3. 输入验证，防止SQL注入
