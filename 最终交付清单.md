## 4. 构建与运行说明

### 4.1 环境要求
- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34
- Gradle 8.2

### 4.2 构建步骤

```bash
# 1. 进入项目目录
cd /workspace/ShapeCodeMaster

# 2. 使用Gradle构建
./gradlew assembleDebug

# 3. 安装到设备
./gradlew installDebug

# 4. 运行测试
./gradlew test
```

### 4.3 生成Release APK

```bash
./gradlew assembleRelease
```

APK输出路径: `app/build/outputs/apk/release/app-release.apk`
