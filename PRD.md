# 形码打字速度练习App — 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
**形码大师** (ShapeCode Master)

### 1.2 产品定位
专注于形码输入法（五笔、郑码、仓颉等）拆字思维训练的安卓应用，帮助用户系统性提升形码输入能力。

### 1.3 目标用户
- 形码初学者：需要学习字根和拆字规则
- 进阶练习者：希望提升打字速度和准确率
- 专业录入员：需要保持和突破输入速度

## 3. 用户故事

### US-001 单字练习
作为形码初学者，我希望按字根区域练习单字，以便系统性地掌握字根分布。

**验收标准：**
- 可选择横区/竖区/撇区/捺区/折区
- 练习时显示目标汉字
- 输入错误时提示正确字根
- 显示拆字可视化

### US-002 看打练习
作为进阶练习者，我希望对照文本进行输入练习，以便提升连续输入能力。

**验收标准：**
- 显示目标文本
- 实时显示输入进度
- 错误字符高亮显示
- 练习结束显示统计报告

### US-003 错字重练
作为用户，我希望自动收集我常错的字并进行针对性练习，以便强化薄弱环节。

**验收标准：**
- 自动记录错误字
- 错字本可查看和管理
- 支持一键发起错字练习

### US-004 字根查询
作为初学者，我希望随时查询字根信息，以便在学习中快速查找。

**验收标准：**
- 按区位显示字根表
- 支持搜索字根
- 显示字根对应的键位

## 4. 交互流程

### 4.1 主界面结构
```
主界面 (Bottom Navigation)
├── 练习 (Practice)
│   ├── 单字练习
│   ├── 词组练习
│   ├── 句子练习
│   └── 错字本
├── 字根 (Radicals)
│   ├── 字根表
│   └── 字根查询
├── 统计 (Stats)
│   ├── 练习记录
│   └── 数据统计
└── 设置 (Settings)
    ├── 码表选择
    ├── 主题设置
    └── 关于
```

### 4.2 练习流程
1. 用户选择练习类型（单字/词组/句子）
2. 选择具体分类或设置
3. 进入练习界面
4. 显示目标文本和输入区域
5. 用户输入，系统实时反馈
6. 练习结束，显示统计报告
7. 可选择保存到错字本或继续练习

## 5. UI/UX 关键要求

### 5.1 视觉设计
- 采用 Material3 设计规范
- 支持浅色/深色主题
- 练习界面简洁，减少干扰
- 错误提示醒目但不刺眼

### 5.2 交互设计
- 逐键即时反馈，延迟 < 50ms
- 键盘区域与真实键盘布局一致
- 拆字可视化动画流畅
- 支持横屏/竖屏切换

### 5.3 无障碍
- 支持系统字体大小设置
- 支持 TalkBack 屏幕阅读器

## 6. 非功能性需求

| 需求 | 描述 |
|------|------|
| 性能 | 练习界面响应延迟 < 50ms，码表查询 < 10ms |
| 兼容性 | Android 8.0+ (API 26+) |
| 存储 | 码表数据压缩存储，总包体积 < 50MB |
| 离线 | 核心练习功能完全离线可用 |
| 安全 | 用户数据本地加密存储 |

## 7. 数据需求

### 7.1 码表数据
- 五笔86版：约6763个汉字 + 常用词组
- 五笔98版：约6763个汉字 + 常用词组
- 字根表：25个键位 × 平均10个字根

### 7.2 练习内容
- 单字：按字根区域分类
- 词组：常用二字、三字、四字词组
- 文章：经典赛文、日常文章

### 7.3 用户数据
- 练习记录：时间、类型、速度、准确率
- 错字本：汉字、错误次数、最近错误时间
- 设置：码表选择、主题、难度
