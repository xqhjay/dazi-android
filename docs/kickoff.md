# 项目启动文档

## 项目基本信息

| 项目 | 内容 |
|---|---|
| 项目名称 | 打字提速 Android App |
| 项目负责人 | AI 项目经理 |
| 启动日期 | 2026-06-20 |
| 目标交付物 | Android APK（arm64-v8a）+ 测试报告 + 项目文档 |

## 项目目标

将开源网页项目 [limalivy/zjxx](https://github.com/limalivy/zjxx) 的汉字下落式打字玩法，改造为一款商业级 Android 移动应用。去除原项目的修仙化包装，采用现代极简专业风格，面向希望提升中文输入速度的用户。

## 项目范围

### 包含范围

- 核心练习模式：限时挑战、自由练习、无尽模式
- 扩展功能：每日挑战、成就徽章、统计报表、错字重练、自定义字集
- 个性化：深色/浅色主题切换、音效、震动反馈
- 数据：离线本地排行榜、SQLite 持久化
- 平台：Android arm64-v8a

### 不包含范围（V2）

- 云端同步与账号系统
- 多人在线排行榜
- iOS 版本
- 词组/句子练习

## 技术栈

- 前端：Vue 3 + Vite + TailwindCSS
- 后端/桥接：Tauri 2 + Rust
- 数据库：SQLite（tauri-plugin-sql）
- 构建：Tauri Android

## 风险与应对

| 风险 | 影响 | 应对策略 |
|---|---|---|
| Tauri 2 Android 本地构建失败 | 高 | 启用 GitHub Actions 远程构建 |
| 中文字符输入兼容性问题 | 中 | 充分测试主流输入法 |
| 低端设备性能不足 | 中 | 提供渲染降级方案 |
| 功能范围蔓延 | 中 | 严格按照 MVP 验收标准控制 |

- 目标设备为 Android 7.0 及以上 arm64 设备
