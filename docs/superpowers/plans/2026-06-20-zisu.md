# 字速 (ZiSu) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一个 Tauri 2 (Rust) + React 的中文打字速度练习 Android App（arm64-v8a），双模式玩法 + 统计/掌握度/打卡/成就，GitHub Actions 构建并上传 gofile.io。

**Architecture:** Tauri 2 移动端，Rust 后端用 rusqlite 持久化 + serde 命令暴露；React 18 + Vite + TS 前端，Zustand 状态，Tailwind 样式，自绘 SVG 图表。字集 JSON 打包进 assets。

**Tech Stack:** Tauri 2, Rust, rusqlite, serde, React 18, Vite, TypeScript, Zustand, Tailwind CSS, Vitest, GitHub Actions
