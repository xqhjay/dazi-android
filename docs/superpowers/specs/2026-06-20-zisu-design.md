- 平台: Android (arm64-v8a only)
- 技术栈: Tauri 2 (Rust) + React 18 + Vite + TypeScript

- 定位: 纯工具型中文打字速度练习器（去除修仙包装）

## 3. 核心玩法（双模式）

首页 4 个入口：

| 模式 | 显示机制 | 说明 |
|---|---|---|
| 限时挑战 | 下落式 | 4 分钟，生命值，速度等级 Lv.1→8，通关升级 |
| 自由练习 | 下落/行式可切换 | 不限时，自定义速度 0.1–5.0×，错字智能复现 |
| 无尽模式 | 下落式 | 初始 0.5×，每 15s +0.1×，最高分记录 |
| 打字测试 | 行式 | 固定段落逐字输入，精确测 WPM/准确率 |

- 下落式：字符从顶部下落，底部输入栏 + 系统拼音 IME 输入消除
- 行式：一行字符依次高亮当前字，逐字输入，传统打字测试范式

## 4. 字集与数据层

- 内置 3 字集（按字频排序）：常用字 1-500 / 501-1000 / 1001-1500
- 字集 JSON 打包进前端 assets，启动时注册到 Rust
- 各字集的等级、最高分、掌握度相互独立

## 5. 扩展功能

1. 统计仪表盘：WPM 趋势、准确率趋势、练习时长、按字集维度统计（自绘 SVG）
2. 单字掌握度追踪：每字记录对/错次数与掌握度评分，错字高概率复现
3. 每日目标与连续打卡：每日目标（字数/时长），连续打卡天数，日历热力图
4. 成就 + 自定义 + 主题 + 触感：成就徽章；自定义练习（勾选特定字）；深/浅主题；正确/错误震动反馈

## 6. 移动端 UI/UX（简洁工具型）

- 底部 Tab 导航：练习 / 统计 / 字集 / 我的（拇指友好区）
- 风格：Notion/Linear 风——大留白、低信息密度、系统字体、克制动效
- 安全区：适配刘海/挖孔，输入栏避开键盘
- 配色：中性灰为主 + 单一强调色（靛蓝），深浅双主题

## 7. Rust 后端 + 持久化（SQLite）

表设计：
- `practice_sessions`（会话记录：模式/字集/时长/WPM/准确率）
- `char_mastery`（字集+字 → 对错次数/掌握度/最后出现时间）
- `daily_stats`（日期+字集 → 练习次数/字数/时长）
- `streaks`（连续打卡/最长记录/最后活跃日）
- `achievements`（成就解锁记录）
- `settings`（主题/字集/触感等 KV）
- `high_scores`（模式+字集 → 最高分）

Rust 暴露类型化 `#[tauri::command]`：记录会话、查询统计、读写字集掌握度、成就检查等。

## 8. 构建与分发

- `tauri.conf.json` 限制 `abiFilters: ["arm64-v8a"]`
- GitHub Actions 工作流：安装 Rust + Android SDK/NDK + Node → `tauri android build` → 产物 APK → `curl` 上传 gofile.io → 输出下载链接
- 工作流文件随仓库提供，push 即可触发

## 9. 测试策略

- Rust 单元测试：数据层、统计聚合、掌握度评分算法
- 前端单元测试（Vitest）：下落/行式游戏逻辑、WPM 计算、错字复现
- 边界 QA：按 tauri-edge-case-qa 原则审查 IME 中断、生命周期、安全区、内存泄漏

## 11. 智能错字复现算法

- 错字/漏字标记为"待强化"，加入强化池
- 10 秒后从强化池高概率（70%）抽取复现
- 连续正确输入 2 次后从强化池移除，恢复正常概率
- 掌握度评分 = correct / (correct + wrong)，阈值 0.85 视为掌握
